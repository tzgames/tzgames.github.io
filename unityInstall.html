In this tutorial, I am going to show you how to install and run unity!!

Step One:

Go to this website and download UnityHub for the operating system you are currently using.

^ Click on the text above ^


Step Two:

Click on the UnityHubSetup.exe file that will appear at the bottom of your screen and follow the instructions it gives you to install.


Step Three:

Once it has finished downloading, open up unity hub. If you haven't already, create your unity ID.


Step Four:

Once you have done all this, head back to the unity website and select the box labelled Choose Your Unity Version. This will open up the unity download archive where you can select the version of unity you want to install.


Step Five:

Finally click the download button for the version you want to install which will take you to unity hub. Follow the on-screen instructions to download the unity editor.
